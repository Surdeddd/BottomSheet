# Recipes

Copy-paste-ready patterns built on `@surdeddd/bottom-sheet`. Each recipe is
self-contained, framework-typed (React unless noted), and links back to the
relevant adapter docs.

| Recipe | What it solves |
| --- | --- |
| [Image gallery](./image-gallery.md) | Pinterest-style grid in a sheet, swipe-up to fullscreen |
| [Filter panel](./filter-panel.md) | Always-visible filter UI with `setAllowed` no-close mode |
| [Comment thread](./comment-thread.md) | Inner-content scroll vs. drag, keyboard-aware input dock |
| [Checkout drawer](./checkout-drawer.md) | Cancellable close guard via `before-snap` |
| [Navigation drawer](./navigation-drawer.md) | Material You left drawer with peek pull-tab + hamburger |
| [Media mini-player](./media-mini-player.md) | Always-visible Spotify-style player, drag-up for full controls |
| [Comment composer](./comment-composer.md) | Auto-expand on input focus, keyboard-aware send button |
| [Wallet: send money](./wallet-send-money.md) | Multi-step financial flow with `before-snap` guard + back-button |
| [Programmatic drag](./programmatic-drag.md) | `engine.dragTo(px)` for previews, scrubs, animated entrances |
