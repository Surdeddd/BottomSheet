# React example — `@surdeddd/bottom-sheet`

Minimal React + Vite playground for the `BottomSheet` adapter. Demonstrates
three snap points (`minimized`, `half`, `full`) with spring physics, focus trap,
and Escape-to-close.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173.

## Files

- `src/App.tsx` — single component using `<BottomSheet>` from `@surdeddd/bottom-sheet/react`.
- `src/main.tsx` — Vite entry, mounts to `#app`.
- `index.html` — page shell with a tiny inline stylesheet.
