# Vue 3 example — `@surdeddd/bottom-sheet`

Minimal Vue 3 + Vite playground for the `BottomSheet` adapter. Demonstrates
three snap points (`minimized`, `half`, `full`) with spring physics and
Escape-to-close.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173.

## Files

- `src/App.vue` — single SFC using `<BottomSheet>` from `@surdeddd/bottom-sheet/vue`.
- `src/main.ts` — Vite entry, mounts to `#app`.
- `index.html` — page shell with a tiny inline stylesheet.
